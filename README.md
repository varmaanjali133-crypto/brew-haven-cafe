# ☕ Brew Haven

Brew Haven is a modern café website built using HTML, CSS, and JavaScript.

## Features

* Responsive Design
* Interactive Menu Search
* Category Filters
* Wishlist System
* Shopping Cart Functionality
* Quick View Modal
* About Page
* Gallery Page
* Contact Page
* Modern UI/UX Design

## Technologies Used

* HTML5
* CSS3
* JavaScript
